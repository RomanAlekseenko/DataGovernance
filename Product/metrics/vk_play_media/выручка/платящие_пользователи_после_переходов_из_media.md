# Платящие пользователи после переходов из Media

## Бизнес-определение
Метрика показывает количество уникальных пользователей, которые после перехода из материалов VK Play Media совершили покупку в продуктовом контуре VK Play по согласованному правилу атрибуции.

## Зачем используется
* Для оценки числа монетизирующихся пользователей после переходов из Media.
* Для отделения денежного эффекта от простого роста трафика и переходов.
* Для расчета ratio-метрик revenue-контура.

## Условия расчета
Метрика считается как количество уникальных пользователей, у которых после перехода из материала Media произошла хотя бы одна атрибутированная покупка.

### Включаются
* Пользователи с хотя бы одной валидной атрибутированной транзакцией после перехода из Media.

### Не включаются
* Пользователи без подтвержденной покупки.
* Пользователи с транзакциями, не атрибутированными переходу из Media.

## Как интерпретировать изменения
Рост:
Рост означает увеличение числа пользователей, которые после переходов из Media доходят до покупки.

Падение:
Падение означает снижение числа платящих пользователей, атрибутированных Media.

## Ограничения и предостережения
* Метрика зависит от качества user matching между Media и транзакциями.
* Один пользователь с несколькими покупками считается один раз.
* Без фиксированного правила атрибуции значение не является воспроизводимым.

## Минимальные проверки качества
* Для атрибутированных покупок доступен стабильный user key.
* Значение платящих пользователей не превышает число пользователей с переходами.
* Метрика сверяется с выручкой и числом транзакций в том же атрибуционном контуре.
