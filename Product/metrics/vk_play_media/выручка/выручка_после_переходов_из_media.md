# Выручка после переходов из Media

## Бизнес-определение
Метрика показывает сумму выручки, полученной в продуктовом контуре VK Play от пользователей, которые перешли туда из материалов VK Play Media по согласованному правилу атрибуции.

VK Play Media

## Зачем используется
* Для оценки прямого revenue-эффекта Media на продуктовый контур.
* Для сравнения Media не только по переходам, но и по денежному результату.
* Для анализа того, приводит ли трафик из материалов к монетизируемому действию.

## Условия расчета
Метрика считается как сумма выручки по транзакциям, атрибутированным переходам из материалов VK Play Media.

Зафиксированное рабочее правило:
* базовый сценарий атрибуции опирается на переход из материала Media в продуктовый контур;
* окно revenue-attribution для direct-сценария составляет `3` дня;
* revenue scope включает покупки в любом другом домене экосистемы.

### Включаются
* Транзакции, которые можно надежно атрибутировать переходу из материала Media.

### Не включаются
* Транзакции без надежной связи с переходом из Media.
* Технические, тестовые и отмененные транзакции, если это будет зафиксировано в финальном правиле.

## Как интерпретировать изменения
Рост:
Рост означает усиление денежного эффекта Media на продуктовый контур.

Падение:
Падение означает снижение монетизируемого эффекта переходов из Media.

## Ограничения и предостережения
* Метрика критически зависит от выбранного окна атрибуции.
* Даже при наличии перехода метрика не доказывает полный причинно-следственный эффект Media.
* Без правила возвратов и корректировок значение может быть нестабильным.

## Минимальные проверки качества
* Для каждой транзакции доступен идентификатор и денежная сумма.
* Атрибутированные транзакции не дублируются между сценариями расчета.
* Значение сверяется с контрольной транзакционной отчетностью.
