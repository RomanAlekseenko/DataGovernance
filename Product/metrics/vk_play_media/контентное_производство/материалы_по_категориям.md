# Материалы по категориям

## Бизнес-определение
Метрика показывает распределение опубликованных материалов VK Play Media по категориям за выбранный период.

## Домен
VK Play Media

## Сквозная метрика или нет
Нет

## Владелец
Руководитель отдела аналитики и данных.

## Зачем используется
* Для контроля тематического баланса контентного плана.
* Для анализа структуры выпуска материалов.
* Для сопоставления объема публикаций и эффективности по категориям.

## Источник истины
CMS.

## Гранулярность
* по дням;
* по неделям;
* по месяцам;
* по категориям.

## Частота обновления
Ежедневно до 10:00.

## Условия расчета
Метрика считается как число опубликованных материалов в разрезе категорий за выбранный период.

Зафиксированное правило:
* если материалу назначено несколько категорий, он учитывается во всех назначенных категориях.
* метрика считается по публикациям, а не по материалам, получившим просмотры.

### Включаются
* Материалы с валидной категорией из согласованного справочника.
* Материалы из URL-контура `news` учитываются как `новости`.
* Материалы из URL-контура `secret` учитываются как `руководства`.
* Материалы из URL-контура `feat` учитываются как все остальные бизнес-типы без отдельного URL-контура, включая `статьи`, `обзоры`, `интервью`, `подборки`, `спецпроекты` и `партнерские публикации`.
* Материалы форматов `интервью` и `подборки` учитываются наравне с остальными публикациями домена.
* Материалы формата `спецпроект` учитываются наравне с остальными публикациями домена.
* Партнерские публикации учитываются наравне с остальными публикациями домена.
* Видео и стримы учитываются наравне с остальными публикациями домена.

### Не включаются
* Материалы без категории или с временными техническими тегами вместо бизнес-категории.

## Как интерпретировать изменения
Рост:
Рост доли категории означает усиление редакционного фокуса на данном тематическом направлении.

Падение:
Падение доли категории означает ослабление тематического фокуса или перераспределение редакционного ресурса.

## Ограничения и предостережения
* При нескольких категориях один и тот же материал будет входить в несколько категориальных срезов.
* Метрика описывает структуру выпуска, но не показывает качество или спрос.
* Смена справочника категорий ломает сопоставимость периодов.
* Материалы без новых публикаций в отчетном периоде не входят в расчет, даже если получили просмотры.
* Без согласованного категорийного справочника из CMS метрика может по-разному считаться разными командами.

## Минимальные проверки качества
* Категории материалов соответствуют согласованному справочнику.
* Для мультикатегорийных материалов зафиксировано правило учета во всех назначенных категориях.
* Доля материалов без категории контролируется.
