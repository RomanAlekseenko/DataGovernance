## Назначение документа
Документ описывает рекомендуемую структуру системы дашбордов для домена VK Play Media по уровням пользователей:
* C-level;
* продуктовые менеджеры;
* аналитики.

Цель карты:
* развести управленческие и диагностические сценарии;
* избежать дублирования метрик в разных отчетах без изменения их смысла;
* зафиксировать, какие метрики и разрезы нужны на каждом уровне.

## Принципы построения системы дашбордов
* На всех уровнях используется единый словарь метрик и единые формулы расчета.
* Каждый следующий уровень детализации раскрывает предыдущий, а не создает альтернативную версию тех же KPI.
* Executive-дашборды показывают только ключевые сигналы и их динамику.
* Дашборды для продуктовых менеджеров должны отвечать на вопрос, какие действия нужно предпринимать.
* Дашборды для аналитиков должны поддерживать диагностику причин, проверку гипотез и контроль качества данных.
